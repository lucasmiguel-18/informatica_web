El HTML

  1. ¿Cómo funciona una página web?

    2. Los equipos conectados a la red se pueden dividir en:

    • Clientes: son los navegadores del usuario.
    • Servidores: son los ordenadores que guardan o almacenan aplicaiones o sitios web.

    Nosotros cuando entramos a un servidor lo que hacemos es enviar una petición para ver la página web y el HTTP (Protocolo de Transferencia de HiperTexto)se encarga de enviarnos una respuesta, que es la página.

  2. El DNS

    3. Los clientes y los servidores tienen ambos una identificación en Internet denominada IP. El DNS sirve para traducir los nombres de las páginas web a su dirección IP.
